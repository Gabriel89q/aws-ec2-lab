# aws-ec2-lab


# AWS EC2 Lab

Projeto criado para praticar fundamentos de computação em nuvem utilizando serviços da AWS.

---

# Objetivo

Realizar o provisionamento de uma instância EC2 e compreender conceitos básicos de:

- EC2
- VPC
- Sub-redes
- Security Groups
- IP Público e Privado
- Regras de acesso HTTP

---

# Serviços AWS Utilizados

- Amazon EC2
- Amazon VPC
- Security Groups
- IPv4 Público

---

# Tipo da Instância

- t3.micro

---

# Sistema Operacional

- Amazon Linux 2023
